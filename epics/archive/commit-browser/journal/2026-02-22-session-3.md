# Session 3 — 2026-02-22

## What was done

### File view improvements

1. **FAB navigates between hunks** — Up/down FAB buttons jump between groups of consecutive changes, not individual lines.

2. **Changed line highlighting** — Background colors on changed lines matching diff view: green (added), red (deleted), blue (modified).

3. **Full row tappable** — Click handler moved from 4px gutter to entire `<tr>` row for mobile usability.

4. **Hunk-level toggle (tig-style)** — Tapping any line in a hunk toggles ALL old/deleted lines for that group at once. Expansion block placed after the last line of the hunk. All lines in the group share the same toggle. Pre-processing groups consecutive modified/deleted lines into hunks with shared IDs.

### Font & infrastructure (from session 2 continuation)

5. **Geist Mono** — Replaced all SF Mono/Fira Code/Consolas references with Geist Mono loaded from Google Fonts CDN. 8 files across CSS, HTML, and xterm.js config.

6. **No-cache middleware** — `Cache-Control: no-store` on all `/static/` responses for dev.

7. **text-size-adjust fix** — Prevents mobile browser font boosting on code tables.
