# T1: Basic Wrapper Script

## What was worked on
- T1: Create basic wrapper script with subprocess invocation

## What was done
- Created `merlin-bot/` directory
- Created `merlin-bot/claude-wrapper.py` with:
  - PEP 723 inline dependency metadata (no external deps needed for T1)
  - `ClaudeResult` dataclass with fields: result, session_id, stderr, exit_code, duration, usage, model, raw_output
  - `invoke_claude()` function supporting all required parameters: prompt, caller, session_id, model, allowed_tools, append_system_prompt, skip_permissions, max_turns, max_budget_usd, timeout
  - Always uses `--output-format json` for structured output
  - Always uses `--dangerously-skip-permissions` by default (opt-out via `skip_permissions=False`)
  - Uses `--append-system-prompt` (not `--system-prompt`) to preserve Claude Code tools
  - JSON parsing of Claude output with fallback to raw text on parse failure
  - Error handling for: FileNotFoundError (exit 127), TimeoutExpired (exit 124), non-zero exit, invalid JSON

## Validation
Validated via mock-based tests run inline (not yet formal pytest — that's T6):
1. Command construction: verified all flags appear in correct positions
2. JSON parsing: verified session_id, result, usage, model extracted correctly
3. FileNotFoundError: returns exit_code=127 with clear message
4. TimeoutExpired: returns exit_code=124 with timeout info
5. Non-JSON fallback: raw stdout returned as result
6. Non-zero exit: stderr and exit_code preserved
7. skip_permissions=False: flag correctly omitted

All assertions passed.
